CIRCUITRY AND PROGRAMMING

OVERVIEW

As saturated soil has a drastically different resistance when compared to dry soil, the resistivity can be used as a method of measuring the wetting front. The basic principle of how the electrical wetting front detector works is by measuring the voltage (proportional to resistivity) between two metal strips (sensor) in the soil and lighting up an LED if the resistivity drops below a threshold value (determined during calibration). The circuitry for the electrical wetting front detector can be divided into two main sections.

  • The sensor circuitry to give a reading of soil resistivity.
  • The microcontroller (STM32F103C8 Blue Pill) used to calibrate, read and light up the LEDs to signal the wetting front.

All circuitry and electrical components for the wetting front detector are soldered onto the custom PCB. Relevant files needed to fabricate this PCB can be found at the end of this section.

SENSOR CIRCUITRY

The circuit shown on the left is a simple circuit used to measure the voltage (and hence the resistance) between V1 and GND (representing the two metal strips). The way the circuit operates is as follows:


  • When the two metal strips are in contact with dry soil, the resistance between V1 and GND is very high and the two metal strips are effectively non conductive.
  • Since no current flows, the point V1 is at 3.3V. The micro controller reads this high value and sets the corresponding LED to be off.
  • When the metal strips are in contact with wet soil, the resistance between V1 and GND is low and there is a conductive path between the two metal strips.
  • The microcontroller reads this low value and turns the corresponding LED on, signalling that the wetting front has reached the depth of the metal strips.
  • Note that the capacitor is present to eliminate any AC signals that may interfere with the readings.

PROGRAMMING THE MICROCONTROLLER

The microcontroller used is a STM32F103C8 "Blue Pill". When compared with an Arduino UNO, the Blue Pill has higher performance, lower power consumption and lower cost. There are multiple ways of programming the microcontroller such as using a JTAG adapter, an mbed platform (eg: nucleo board) or a USB to ttl serial adapter. The pinout capabilities of the Blue Pill is shown below

source: http://wiki.stm32duino.com/index.php?title=Blue_Pill

In this guide, we shall take the USB to ttl serial adapter approach. We will be using a FT232RL board as the serial to ttl adapter as a low cost option (available for less than £1). The arduino integrated development environment (IDE) will be used in conjunction with the FT232RL board to program the Blue Pill.

The steps below outline the process of programming the Blue Pill

  • First download the Arduino IDE here
  • Once opened, you will need to add the STM32F103C8 board using the boards manager. Go to Tools > Board > Boards Manager and search for "STM32F103". The expected window is shown below. Click install.
  • Under tools. change the board to "Generic STM32F103C series" and make sure the other settings are as follows:
  • Connect the Blue Pill to the FT232RL board which connects to the computer's USB as shown below, a breadboard is recommended for this. The Tx and RX wires are the serial links and the 5V and GND wires provide power to the Blue Pill. It is also possible to use the 3.3V Vcc by changing the voltage jumper FT232RL board. Make sure that the BOOT 0 jumper is connected to 1 and the BOOT 1 jumper is connected to 0.
  • Under Tools > PORT, select the communication port (COM) that corresponds with the USB connection that the Blue pill is connected to.
  • Insert the code used for the electrical wetting front detector (attached at the end of this section) on the IDE text editor. Compile and upload the code onto the Blue Pill.

BUILD PROCESS

The build plan of the reader module is outlined below.

  • 3D print the box and lid of the reader module. Screw on the nuts and bolts to the 3D with the spring in between the bolt and box. This creates a spring loaded contact point that ensure a stable electrical connection when in use.
  • Attach the wires to the bolts using a second nut. This forms the interface to the sensor module. Although the matching of wire to screw configuration is not critical, it is recommended that the middle screw is used for ground.
  • Solder the components onto the PCB using the diagrams below. Ensure that the Blue Pill microcontroller has been programmed before soldering. If you are unsure if the microcontroller has been prorgammed correctly, you may want to solder sil sockets onto the PCB first. Note that the LEDs, button and switch are on the top side of the PCB and the remaining components are on the bottom side.
  • Solder the wires to the PCB noting the ground wire and insert the batteries into the battery holder.
  • Test the circuit to ensure the soldering is correct and and the circuit is working. This can be done by creating a conductive path between and bolt and the "ground" bolt and observing that a LED turns on.
  • Place the PCB in the box aligning the mounting holes. Put the lid on and screw the box shut

BUILD PROCESS

The build plan of the sensor module is outlined below.

  • Measure and cut the metal strips according to the dimensions specified and drill holes at the marker points shown below. Note that a small area is left at one end to allow for the wire to be crimped.
  • Strip and crimp one end of the wire to the metal strip.
  • Wrap the metal around the PVC tube at spacings as shown below.
  • Attach the metal strips to the PVC tube using rivets. Don't forget to drill an extra hole for each strip just above the crimped wire to route the wire into the tube! The result should look like this:
  • 3D print the cap for the sensor module (relevant file at the end of this section). If a 3D printer is not avaliable, the top cap can be milled using a lathe and hand drill.
  • Screw on the nuts and bolts to the cap and attach the wires to the bolts using a second nut. This forms the interface to the reader module. Although the matching of wire to screw configuration is not critical, it is recommended that the middle screw is used for ground.
  • Test that the metal strips are electrically conductive all the way to the screws on the cap using a multimeter.
  • Optional but recommended: fill the inside of the cap with hot glue or silicon to create a waterproof seal and secure the electrical connection.
